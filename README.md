# Poll_Editor
🎯 Poll Editor - Django Project

A simple Poll Voting Web Application built using Django and Bootstrap.

Users can:

✅ View polls
✅ Vote for choices
✅ See live results
✅ Manage polls using Django Admin

🚀 Technologies Used
Python
Django
SQLite3
HTML
Bootstrap 5

⚙️ Installation
pip install django
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver

🌐 URLs
Page	URL
Polls Page	http://127.0.0.1:8000/polls/
Admin Panel	http://127.0.0.1:8000/admin/

📁 Features
Poll Listing
Voting System
Result Display
Template Inheritance
Bootstrap UI
Django ORM
Admin Dashboard
